The Gutenberg Project's Redesigned Interface

Bridging Web Technology to Open Source Literature


Written by Daniel Phan, 07/02/2014:


This is the re-imagined web design of the Gutenberg project.
The design of the website looks take a modern approach of web design to bring to life an open source text.

For the assignment the three Gutenberg Project that follow part 1 specifications are:

  • 1. Estabrooke's Photography in the Studio and in the Field
  • 2. Kafka's Metamorphsis
  • 3. Shelly's Frankenstien

An extra example was generated using Lewis Carol's Alice in Wonderland (this website design does not follow
specifications). Please refer to the README for more detailed explanations of the assignment fulfillments.


The web design will follow the following specs:

  • [X] Theme paragraphs with CSS
  • [X] Theme header tags with CSS
  • [] Theme images and images captions with CSS
  • [X] Change the colour scheme to something aesthetically pleasing
  • [] Apply 1 CSS file to 3 gutenberg documents.
  • [ ] document in your README what modifications you had to make to the project gutenberg HTML files.
  • [ ] Take 3 screenshots of Firefox running all of your pages that demonstrates your theme.

main.css will be used to template the 3 Gutenberg Projects.
